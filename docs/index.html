OpenWatch

Client-side observability suite. Analyze logs, CPU profiles, distributed traces, and network requests - all in your browser. No backend, no data sent anywhere, 100% private.

Installation

# Docker (Recommended)
docker run -d -p 3000:3000 ghcr.io/rtorreson/openwatch:latest

# Access
http://localhost:3000

# Or clone and run locally
git clone https://github.com/rtorreson/openwatch.git
cd openwatch
npm install
npm run dev
